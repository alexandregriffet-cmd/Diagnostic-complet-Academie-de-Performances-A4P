PACK GITHUB PROPRE A4P

Ce pack est prévu pour repartir de zéro dans un nouveau dépôt GitHub Pages.

Contenu :
- index.html
- synthese.html

Important :
- il ne faut garder QU'UN SEUL fichier nommé exactement index.html à la racine
- ne pas conserver de fichiers du type "index 8.html" ou "index 10.html"

Étapes :
1. Crée un nouveau dépôt GitHub
2. Supprime tout fichier inutile
3. Envoie uniquement les fichiers de ce pack
4. Active GitHub Pages sur la branche main / root
